# InclusiWeb

InclusiWeb is a Chrome extension that transforms web pages into an accessible reading environment. The project focuses on improving readability through customizable accessibility features such as text resizing, dyslexia-friendly typography, high contrast mode, text-to-speech, and reading assistance.

---

## Features

- Reader Mode
- Adjustable font size
- Dyslexia-friendly font mode
- High contrast mode
- Text-to-Speech
- Hover-to-read
- Paragraph highlighting during reading
- Responsive and accessible interface

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Chrome Extension API (Manifest V3)
- Web Speech API

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-user/inclusiweb.git
```

2. Open Chrome.

3. Go to:

```
chrome://extensions
```

4. Enable **Developer Mode**.

5. Click **Load unpacked**.

6. Select the project folder.

---

## Usage

1. Open any webpage.
2. Click the InclusiWeb extension icon.
3. Select **Open Reader**.
4. Use the accessibility toolbar to customize the reading experience.
